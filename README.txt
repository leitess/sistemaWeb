Sistema Web
===========

Projeto desenvolvido para a disciplina: laboratorio de desenvolvimento multiplataforma. Onde
o objetivo será cadastrar e listar pessoas e seus respectivos endereços.

Este projeto está usando as bibliotecas: Express e Handlebars. Para instalar use o comando:
      npm install

Para rodar use o comando:
      npm run dev

As rotas são:
    GET: Para cadastrar, a endpoint é / -> http://localhost:8081/
    GET: Para listar, a endpoint é /list -> http://localhost:8081/list
